“在Mac下搭建Apache服务器”

在Mac下搭建Apache服务器
1,前往–>个人
[id]: /img/1.png “img1”


2,（2）在你的个人目录下新建一个文件夹，改名为“Sites”(改完后可见到文件夹中有个指南针的图标)。注意要放在个人用户的文件夹下，因为它和桌面、文稿等文件夹一样是你私人的文件夹，这个文件夹用来专门给外部网络访问的。
[id]: /img/2.png “img2”


3,打开终端，控制台输入下面代码

$cd /etc/apache2
[id]: /img/3.png “img3”


4,第三步(可选):

备份文件，以防不测，只需要执行一次就可以了

$sudo cp httpd.conf httpd.conf.bak

如果操作失误，可输入下面命令恢复

$ sudo cp httpd.conf.bak httpd.conf

5,进入编辑模式,输入如下代码,会提示输入密码,输入后回车
$sudo vim httpd.conf
[id]: /img/4.png “img4”


[id]: /img/5.png “img5”


6,输入 “/DocumentRoot” 后按回车(切换英文状态)

[id]: /img/6.png “img6”


[id]: /img/7.png “img7”


7,直接输入”i”(不敲回车)底部会出现 “–insert–”的标志证明进入编辑模式
[id]: /img/8.png “img8”


8,将DocumentRoot和Directory后面两个路径修改为

例：
DocumentRoot “/Users/你的用户名/Sites”

Directory “/Users/你的用户名/Sites”

修改完成后按ESC退出编辑模式,滑倒最底部

9,搜索PHP,操作:直接输入 /php 回车如下图
如果搜到php所在的那一行前面带”#”符号,先按数字’0’，这样光标会移到行首；再按字母’x’，此时前面的注释’#’就没有了。
[id]: /img/9.png “img9”


10,此时可以输入”:wq”回车保存修改后退出了。(如果觉得修改出错了，可以输入”:q”不保存修改直接退出，再重新来过)。

11，保存完毕后输入下面代码切换工作目录

$cd /etc
12,拷贝php.ini文件

$sudo cp php.ini.default php.ini

[id]: /img/10.png “img10”


13,重新启动apache服务器

$sudo apachectl -k restart

14,如果提示以下错误是正常的：

httpd: Could not reliably determine the server’s fully qualified domain name, using teacher.local for ServerName

httpd not running, trying to start
